# Argon Portfolio Website

A clean, minimalist personal portfolio website for Premsagar Kumar (aka Argon).

## Features

- Clean, minimalist design
- Responsive layout
- Dark/Light theme toggle (with localStorage persistence)
- Social media links
- About section

## Setup

1. Open `index.html` in a web browser
2. For the profile picture, add an image file named `profile.jpg` in the root directory, or update the image source in `script.js`

## Customization

- Update social media links in `index.html`
- Modify colors and styles in `styles.css`
- Add your profile picture as `profile.jpg` or update the image source

## Files

- `index.html` - Main HTML structure
- `styles.css` - Stylesheet
- `script.js` - JavaScript for theme toggle and interactions
